# Features

-   Authentication with bcrypt and jsonwebtoken
-   Emialing for support
-   Add/Remove Friends
-   Live Chat with freinds using websocket
-   Image uploading
-   Create recipes
-   Publish recipes as post
-   Post view as carosule
    -   Image and description view
    -   Ingredients table
    -   Instructions
    -   Rating 
    -   Comments
    -   Adding recipe from friends
-   Profile Page
-   Public Page
